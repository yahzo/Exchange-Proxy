# Exchange-Proxy

Ce depot est organise par etapes pour le projet d'application temps reel.

## Branches

- `step-1` : correction du Kafka Kata demande en prerequis.
- `step-2` : application proxy de taux de change en Java avec Kafka, Elasticsearch et Kibana.

## Utilisation

Pour consulter la correction du Kafka Kata :

```bash
git checkout step-1
```

Pour lancer l'application proxy :

```bash
git checkout step-2
```

Chaque branche contient son propre README avec les commandes de lancement.
